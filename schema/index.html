Documentation

Schema Overview

The genealogy schema includes the following main sections:

  • metadata - Project information and source details
  • people - Individual person records (schema.org/Person)
  • families - Family relationships and marriages
  • places - Geographic locations (schema.org/Place)
  • sources - Bibliographic sources and citations

WikiTree ID Format

Records use WikiTree IDs in the format: Surname-Number

Example: England-1357, Duncan-166
