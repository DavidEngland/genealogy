Parse a New GEDCOM File:
   php schema/gedcom_parser.php GEDs/myfile.ged data/myfile.json

Export to CSV (Lookup):
   php schema/csv_exporter.php data/myfile.json output.csv lookup

Export to CSV (Full):
   php schema/csv_exporter.php data/myfile.json output.csv full

Process All GEDCOM Files:
   schema/batch_process.sh

================================================================================
                           TROUBLESHOOTING
================================================================================

Q: Can't open HTML file?
A: Use: open schema/index.html
   Or drag file to your browser

Q: CSV export is empty?
A: Make sure you:
   1. Loaded a JSON file first
   2. Selected "Export Lookup CSV" or "Export Full CSV"
   3. File has data (check JSON with text editor)

Q: Want to add more GEDCOM files?
A: Just run:
   php schema/gedcom_parser.php GEDs/newfile.ged data/newfile.json

Q: How to combine all lookup files?
A: Run:
   cat data/*-lookup.csv > all-people.csv
   Now you have a master index of 19,000+ ancestors
