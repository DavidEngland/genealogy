# WikiTree HTML Markup Tags Style Guide

**Language:** [en](/) | [de](/wiki/Help:De:Empfohlene_Tags) | [fr](/wiki/Help:Fr:Balises_Recommand%C3%A9es) | [nl](/wiki/Help:Nl:Aanbevolen_Tags)

*This page is about HTML markup tags. See [G2G Tags](/wiki/G2G_Tags) for the kind you [follow](/wiki/Special:Following).*

This page is for advanced members. Many members only need to use the formatting options that are available in the buttons above an editing box. Members who want to learn more should start at [Editing Tips](/wiki/Help:Editing_Tips).

The following is **not** a [wiki markup "cheatsheet"](http://en.wikipedia.org/wiki/Help:Cheatsheet) and it is **not** intended to help you use these tags. It is meant to help you distinguish between tags that are recommended and supported (those you see below) and tags that are not recommended on WikiTree (those you don't see below). Before using any tag you don't see below, read [HTML and Inline CSS](/wiki/Help:HTML_and_Inline_CSS) very carefully.

To use this list, start with a tag you're uncertain about and see if it's listed below. Tags and codes are grouped by the character they begin with. For example, if you want to know if `<font>` is recommended, look for it in the Angle Brackets section. If it's not there (it's not) that means it's not recommended.

Ellipses (...) in a tag or code mean the content in that space is variable. The entire tag or code is not necessarily listed below. Modifiers and properties are not included.

Inclusion in this list does not mean the tag is appropriate in every circumstance. For example, the recommendation for [extended quotations](/wiki/Help:Quotations) is to use `<blockquote>`. Manual indentation with a colon (:) is not recommended for quotations but it is recommended for other spacing needs.

If you're still confused about whether you should use a particular tag, just [ask about it in G2G](https://www.wikitree.com/g2g). Tag your question with "style".

## Ampersands

- `&...` such as `&mdash;` or `&#37;` - used to create or replace special characters in links or text - see [http://www.ascii.cl/htmlcodes.htm](http://www.ascii.cl/htmlcodes.htm)

## Angle Brackets

- `<!--...-->` - hidden comment
- `<blockquote>...</blockquote>` - see [Quotations](/wiki/Help:Quotations)
- `<br>` - line break - see [Editing Tips](/wiki/Help:Editing_Tips)
- `<center>...</center>` - see [Editing Tips](/wiki/Help:Editing_Tips), not for styling purposes
- `<includeonly>...</includeonly>` - used in templates - see [Templates](/wiki/Help:Templates)
- `<noinclude>...</noinclude>` - see [Templates](/wiki/Help:Templates)
- `<nowiki>...</nowiki>` - see [http://en.wikipedia.org/wiki/Help:Wiki_markup](http://en.wikipedia.org/wiki/Help:Wiki_markup)
- `<onlyinclude>...</onlyinclude>` - see [Templates](/wiki/Help:Templates)
- `<ref>...</ref>` - see [Sources](/wiki/Help:Sources)
- `<references />` - see [Sources](/wiki/Help:Sources)
- `<sub>...</sub>` and `<sup>...</sup>` - see [Editing Tips](/wiki/Help:Editing_Tips)

## Apostrophes

- `''...''` - italic uses two apostrophes
- `'''...'''` - bold uses three apostrophes
- `'''''...'''''` - bold and italic uses five apostrophes - see [Editing Tips](/wiki/Help:Editing_Tips)

## Asterisks

- `*` - bulleted list
- `**` - nested bulleted list inside a list (level 2 list)
- `***` - nested bulleted list (level 3 list) - see [Editing Tips](/wiki/Help:Editing_Tips)

## Braces/Curly Brackets

- `{|...` - start of wikitable - see [Editing Tips](/wiki/Help:Editing_Tips)
- `{{#...}}` - call advanced parser functions - sysops, see [Docs:Parser Functions](/wiki/Docs:Parser_Functions)
- `{{...}}` - call template or magic word - see [Templates](/wiki/Help:Templates)

## Brackets/Square Brackets

- `[...]` - link to any URL - see [Adding Links](/wiki/Help:Adding_Links)
- `[[...]]` - link to WikiTree [profile](/wiki/Help:Profiles)
- `[[:Category:..]]` - link to WikiTree category - see [Categorization](/wiki/Help:Categorization)
- `[[Category:..]]` - categorize a page - see [Categorization](/wiki/Help:Categorization)
- `[[Image:...]]` - displays an image - see [Photos FAQ](/wiki/Help:Photos_FAQ)
- `[[Project:...]]` - link to WikiTree [project page](/wiki/Help:Projects)
- `[[Space:...]]` - link to WikiTree [free-space profile](/wiki/Help:Free-Space_Profile)
- `[[Wikipedia:...]]` - link to Wikipedia article
- `[[Google:...]]` - link to Google search

## Colons

- `:...` - indent (must be first character on line)
- `::...` - double indent
- `:::...` - triple indent - see [Editing Tips](/wiki/Help:Editing_Tips)

**Note:** Do not indent every paragraph in a biography. If the community decides that wider margins are preferred, this would be done site-wide.

## Equal Signs

- `==...==` - level 2 headline
- `===...===` - level 3 heading
- `====...====` - level 4 heading - see [Editing Tips](/wiki/Help:Editing_Tips)

## Hash/Number/Pound Signs

- `#...` - numbered list item
- `##...` - nested numbered list item (level 2 numbered list)
- `###...` - level 3 numbered list - see [Editing Tips](/wiki/Help:Editing_Tips)

## Hyphen/Minus Signs

- `----` - horizontal dividing line - see [Editing Tips](/wiki/Help:Editing_Tips)
